[Tangram Grid] 简介

==============================

Tangrid Grid是基于Tangram 的一个Grid的控件，特点:
*易于使用
*功能强大且可拔插
*非常方便的扩展出新功能
*性能优异
*能兼容IE6。

如果你想进一步了解Tangram grid 检出项目中的docs/demo。
